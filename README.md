### serialExperimentsNix
This is my personal /etc/nixos directory. Yes, the name is a Lain reference.

### Tree Hiearchy
```
├── flake.nix                   entry point, defines inputs and exposes the nixos configuration
├── flake.lock                 pinned versions of all inputs, guarantees reproducibility
├── README.md                  documentation
│
├── home/                      home manager configuration
│   └── home.nix               user level config, applies only to sidharthify’s session
│
├── nixos/                     system level configuration
│   ├── configuration.nix      the root module that imports every other system module
│   │
│   ├── hardware/              hardware specific modules
│   │   ├── hardware-configuration.nix   autogenerated, defines disks and filesystems
│   │   ├── bluetooth.nix               bluetooth stack
│   │   ├── intel.nix                   intel microcode, firmware, power management
│   │   ├── nvidia.nix                  nvidia driver, kernel modules, opengl integration
│   │   └── sata.nix                    sata controller tuning or quirks
│   │
│   ├── services/              system services that run in the background
│   │   ├── docker-containers.nix       declarative containers
│   │   ├── flatpak.nix                 flatpak daemon
│   │   ├── openrgb.nix                 openrgb hardware control
│   │   ├── pipewire.nix                audio stack
│   │   ├── steam.nix                   gaming subsystem
│   │   ├── tmate.nix                   tmate ssh service
│   │   ├── waydroid.nix                waydroid container setup
│   │   └── zerotier-config.nix         zerotier networking
│   │
│   ├── system/                core operating system configuration
│   │   ├── auto-upgrade.nix           system auto update policy
│   │   ├── bootloader.nix             grub or systemd boot configuration
│   │   ├── desktop.nix                plasma, sddm, display environment
│   │   ├── fonts.nix                  installed font sets
│   │   ├── kernel.nix                 kernel version and parameters
│   │   ├── locale.nix                 locale and language settings
│   │   ├── networking.nix             networkmanager, interfaces, firewall
│   │   ├── nix-settings.nix           nix daemon config, flakes, trusted users
│   │   ├── opengl.nix                 opengl drivers, mesa, acceleration
│   │   ├── openssh.nix                ssh server settings
│   │   ├── printing.nix               cups printing
│   │   ├── time.nix                   timezone
│   │   ├── xdg-portal.nix             xdg portals for flatpak and desktop integration
│   │   └── xkb.nix                    keyboard layout and keymap
│   │
│   └── users/                 system user definitions
│       ├── sidharthify.nix            user account, groups, user packages
│       └── zsh.nix                    shell configuration at system level
│
├── packages/                  custom package set
│   └── packages.nix           local packages exposed as pkgs.<name>
│
└── scripts/
    └── sync-nixos.sh          the sync-nixos script, rebuilds and pushes automatically
```
